```bash
  npx create-react-app .
  npm i -g @aws-amplify/cli@4.24
```

## Run Locally

Clone the project

```bash
  git clone git remote add origin https://github.com/Muhammadqodir-2004/calculator.git
```

Go to the project directory

```bash
  cd yelp
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
